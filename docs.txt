1) Extension for python file is .py
2) Indentation is very important in python it can have atleast one space or atmost 4 space.
3) In the same block each line should have same indentation space.
4) Python has no cammond to declare a variable,  variables are created when you assign a value to it.
5) Comment start with # in python. Variable names are case-sensitive.
6) If you want to specify the data type of a variable, this can be done with casting.
   Example: x = str(3); # '3'
7) You can get the data type of a variable with the type() function.
   Example: x=5; print(type(5)); #int 
8) To create a global variable inside a function, you can use the global keyword.
